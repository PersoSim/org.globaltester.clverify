# Releasetests org.globaltester.clverify

This document describes validation tests that shall be performed on the final product artifacts immediately before publishing. These tests focus on overall product quality and completeness (e.g. inclusion/integration of required features). For a complete test coverage please also refer to tests defined in the according bundles.

Use a setup where the final product is installed in a fresh Eclipse IDE.

1. [ ] __CL Verify__
	- [ ] run a testcase that uses CL Verify to simulate a physical card and this is used for verification through physical card reader
		- [ ] verify the simulated card
	- [ ] run a testcase that uses CL Verify, do NOT put the antenna into field and finish the testcase (GT must not crash)
